CircleProgressBar
=========

介绍
---

带载入动画的环形进度条，可高度配置，支持配置成表盘。

使用
---

**引用:**
```
dependencies {
    ...
    implementation 'io.github.alexmofer.widget:circleprogressbar:1.0.3'
    ...
}
```
